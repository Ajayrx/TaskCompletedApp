<h1>TaskCompleted</h1>

<p>
TaskCompleted is a simple single-screen Android app built while learning
Android development with <b>Kotlin</b> and <b>Jetpack Compose</b>.
</p>

<p>
The app displays a task completion message with an image and text.
The goal of this project is to understand basic Compose layouts,
resources, and theming.
</p>
